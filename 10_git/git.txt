title: Git
desc: 


q: версия
a: 
git --version
#


q: задать свое имя
a: 
git config --global user.name "ваше имя"
#

q: задать почтовый адрес
a: 
git config --global user.email адрес
#


q: to check configuration (user name etc..)
a: 
git config --global --list
#

q: создать открытый и закрытый ключи
a: 
../Users/User/.ssh

программа ssh-keygen
#


q: скопировать ssh ключ в буфер
a: 
clip < ~/.ssh/id_rsa.pub

#

q: проверить успешность аутитентификации на github по ssh ключу
a: 
ssh -T git@github.com
#

q: для чего ветка master
a: 
Обычно в ней находится стабильная версия программы без ошибок. 
Если вы хотите исправить баг, добавить новую функциональность в проект, попробовать какую-то технологию, 
но не хотите сломать код в основной ветке, вы ответвляетесь из master и трудитесь в своей новой ветке. 
#

q: создать ветку 
a: 
git checkout -b имя-новой-ветки.
#


q: список веток
a: 
git branch
#

q: переключение на ветку
a: 
git checkout name
#

q: изменить название ветки
a: 
git branch -m старое-имя-ветки новое-имя-ветки
#

q: текущее состояние в ветке
a: 
git status

а именно список с названиями изменённых файлов, если они есть, и укажет на те, которые ожидают записи и сохранения (обычно они выделены красным цветом).
#

q: добавить новый файл в git, все файлы
a: 
git add имя-файла
git add -A
#

q: коммит
a: 
git commit -m "ваше сообщение"
#

q: отправить коммиты в репозиторий
a: 
git push origin название-текущей-ветки

 где origin означает репозиторий, который был склонирован на компьютер, то есть ваш форк.
#

q: пулреквест
a: 
 Это предложение изменить код в репозитории.
#

q: подтянуть изменения из ветки
a: 
git pull repository-name branch-name
#

q: сделать ссылку на репозиторий
a: 
git remote add new-rep-name git@github.com:your-repo.git
#

q: переместить файл в другую папку
a: 
git mv git.txt 10_git
#

q: git просит коментарий при коммите, что нажать
a: 
ESC
:wq
Enter

В первой строке введите комментарий.
Сохраните файл и выйдите из редактора 
(для этого в редакторе по умолчанию (Vim) вам нужно нажать клавишу ESC, ввести :wq и нажать Enter)
